# QuariX - LLM Context File

## Project Overview

QuariX is a modern web application built with Next.js 15, React 19, TypeScript, and Tailwind CSS. It provides professional and high-quality website AI agents, Chatbots, iOS, and Android application design development services.

**Live URL**: quarix-gamma.vercel.app
**Repository**: https://github.com/Gokulakrishnxn/QuariX

## Tech Stack

- **Framework**: Next.js 15.5.9 (App Router)
- **UI Library**: React 19
- **Language**: TypeScript 5.7.2
- **Styling**: Tailwind CSS 3.4.17
- **UI Components**: shadcn/ui (New York style)
- **Theme Management**: next-themes 0.4.6
- **Icons**: Lucide React
- **Animation**: tailwindcss-animate

## Key Features

### 1. Theme System
- **Dark Mode**: Pure black background (#000000)
- **Light Mode**: White background
- **Theme Toggle**: Circular button with sun/moon icons
- **Sound Effect**: Click sound on theme toggle
- **Logo Switching**: Different logos for light/dark modes
  - Light mode: Qdarktxtlogo.png
  - Dark mode: QLighttxtlogo.png

### 2. Navigation Bar
- Sticky navigation with backdrop blur
- Responsive design with mobile menu
- Navigation links: Home, Products (dropdown), About, Contact
- Theme toggle button
- "Get Started" CTA button
- Logo that switches based on theme

### 3. Hero Section
- Left-aligned content layout
- Headline: "Plan. Build. Ship." (white text)
- Service description
- "Responsive Design - Optimized for all devices" feature highlight
- Two CTA buttons: "Get Started" and "Learn More"
- Right side: Theme-aware hero image
  - Light mode: darkanime.jpg
  - Dark mode: lightanime.jpg
- Background: Subtle blueprint grid pattern with golden ratio spiral overlay

### 4. Background Patterns
- Blueprint grid pattern (40px grid, very low opacity)
- Thin geometric lines (horizontal and vertical)
- Golden ratio spiral overlay (SVG-based)
- All patterns use monochrome colors with very low opacity
- Futuristic tech aesthetic

## Important Notes

1. **Theme Detection**: Uses `resolvedTheme` to properly detect dark mode even with system preference
2. **Logo Switching**: Logos are swapped (light logo for dark mode, dark logo for light mode)
3. **Image Optimization**: Uses Next.js Image component for optimized loading
4. **Accessibility**: Proper ARIA labels and semantic HTML
5. **Responsive**: Mobile-first design with breakpoints
